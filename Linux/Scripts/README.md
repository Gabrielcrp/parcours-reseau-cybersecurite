# Mini-script Bash : Organiser fichiers

## Objectif
Ce script organise automatiquement les fichiers d'un dossier par extension (txt, jpg, pdf…).

## Comment l'utiliser
1. Placer le script dans le dossier contenant vos fichiers.
2. Ouvrir un terminal dans ce dossier.
3. Lancer le script :
   ./organiser_fichiers.sh
4. Les fichiers seront organisés dans des sous-dossiers selon leur extension.

## Ce que j'apprends
- Utilisation de Linux et du terminal
- Commandes de base : for, mv, mkdir
- Scripting Bash
